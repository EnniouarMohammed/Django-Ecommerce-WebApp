# Installation

```bash
pip install django
```

```bash
virtualenv env
```

# For Mac/ Linux

```bash
env/bin/activate
```

# For Window

```bash
env\scripts\activate
```

```bash
pip install -r requirements.txt
```

```bash
python manage.py makemigrations
```

```bash
python manage.py migrate
```

```bash
python manage.py runserver
```
